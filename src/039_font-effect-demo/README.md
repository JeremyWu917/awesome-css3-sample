# CSS 文本效果

## 文本溢出
> CSS `text-overflow` 属性规定应如何向用户呈现未显示的溢出内容
- `clip` 裁剪
- `ellipsis` 省略号

## 字换行
> CSS `word-wrap` 属性使长文字能够被折断并换到下一行
- `nowrap` 不折断换行
- `break-word` 允许长单词被打断并换行到下一行

## 换行规则
> CSS `word-break` 属性指定换行规则
- `keep-all` 连接字符处换行、保持文本的完整性
- `break-all` 任何字符处换行

## 书写模式
> CSS `writing-mode` 属性规定文本行是水平放置还是垂直放置
- `horizontal-tb` 横向
- `vertical-rl` 纵向
